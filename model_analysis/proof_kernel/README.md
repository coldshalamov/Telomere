# Telomere Proof Kernel

Evidence-level-4 mathematical kernel (see `docs/PROOF_TARGET.md`). It does
not run broad seed searches. Two recurrence kernels live here:

- **v1 / BIT_LITERAL lane** (`entry_state.py` + `viability_search.py`):
  the audited reference kernel — computed freshness, charged discrimination
  channels, earned superposition, selection brackets. Unchanged this
  revision. Artifacts: `best_config.json` (canonical-v1 frontier),
  `bit_literal_target.json` (audited BIT_LITERAL target: 0.1328 %/pass min,
  payback 126 effective, 0.486 @ 500).
- **v-next lane** (`vnext_kernel.py` + `vnext_search.py`): LITERAL_RUN
  re-segmentation, BIT_LITERAL singles, layer-masked expansion (zero
  metadata, fresh = 1, shared tables — law-validated), k-XOR records,
  J2D1/J3D1 jumpstarter profiles, dual window semantics (entry-aligned +
  aB-bit grid with walk-DP clean/dirty pricing). Artifacts:
  `vnext_best.json` (primary: +0.908 %/pass min, payback 81, 0.541 @ 500),
  `vnext_sweep.json`, `vnext_top_profiles.csv`, `charts/`.

Single empirical assumption everywhere: uniform hash law P(match) = 2^-S.
Costs are exact (`costs.py`, mirroring `src/bin/v1_cost_table.rs`); pin
status and golden vectors in `cost_pin_report.json` (cargo was unavailable
in the generating sandbox — re-pin locally).

## Invariants enforced

1. **Freshness is computed, never assumed.** Deterministic search ⇒
   unchanged (content, position-context, depth) ⇒ zero new chances.
   Position-ONLY salting deadlocks (validated: dead by pass 3); the
   layer-indexed mask `expand(seed) XOR mask(layer, offset)` is the
   corrected zero-metadata refresh and sustains (validated:
   `freshness_law_validation.py`).
2. **No uncharged passthrough.** Every output bit is regenerated by a
   record or carried under a charged literal/run header in the emitting
   layer. The legal rechunk is LITERAL_RUN re-segmentation (explicit
   charged boundaries); the uncharged-chunk lanes remain `failed_audit`.
